# Introduction

Welcome to **Kishore UI**. This is a collection of high-performance, accessible, and beautifully animated components that you can copy and paste into your apps.

## Why Kishore Ui?

Most animation libraries are either too heavy or too rigid. Kishore Ui follows the "shadcn" philosophy:

- **Beautiful by default**: Built with modern aesthetics in mind.
- **Accessible**: ARIA compliant and keyboard navigable.
- **Customizable**: You own the code. Change everything.
- **Lightweight**: Only use what you need.

## Inspiration

This project is heavily inspired by:
- [shadcn/ui](https://ui.shadcn.com/)
- [Aceternity UI](https://ui.aceternity.com/)
- [mmk-icons](https://github.com/kishore-sv/mmk-icons)
